1.	Подготовить клиент для тестов
	1.1.	Запустить пустую базу (.cf) дипломной работы в новой конфигурации.
	1.2.	Подождать прогрузки начальной настройки в режиме предприятия.
	1.3.	Закрыть базу.	

2.	Выполнить начальное заполнение
	2.1.	Подключить новую базу к ТестКлиент. В параметрых запуска подключаемой базы указать /N"". 
	2.2.	Запустить сценарий "One". Он выполнит заполнение базы тестовыми данными.
	2.3.	Закрыть базу.

3.	Выполнить тесты менеджера
	3.1.	В параметрых запуска подключаемой базы указать /N"Менеджер". 
	3.2.	Запустить сценарий  "Two". Он выполнит проверку функционала Менеджера.
	3.3.	Закрыть базу.

4.	Выполнить тесты менеджера
	4.1.	В параметрых запуска подключаемой базы указать /N"Специалист". 
	4.2.	Запустить сценарий  "Three". Он выполнит проверку функционала Специалиста.
	4.3.	Закрыть базу.

5.	Выполнить тесты менеджера
	5.1.	В параметрых запуска подключаемой базы указать /N"Бухгалтер". 
	5.2.	Запустить сценарий  "Four". Он выполнит проверку функционала Бухгалтера.
	5.3.	Проверить отчет. Закрыть базу.
