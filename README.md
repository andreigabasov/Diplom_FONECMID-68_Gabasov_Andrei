### 2
Заполните следующие константы:
«Абонентская плата» - заполняется из справочника «Номенклатура» элементом с идентичным наименованием, при необходимости создайте нужную номенклатуру, заполнив все поля;
«Работы специалиста» - заполняется из справочника «Номенклатура» элементом с идентичным наименованием, при необходимости создайте нужную номенклатуру, заполнив все поля;
«Токен управления ТГ ботом» - заполняется токеном бота телеграмм;
«Идентификатор группы для оповещения» - заполняется идентификатором группы телеграмм;

### 3
Заполните регистр сведений «Условия оплаты сотрудников», указав все необходимые данные.

### 4
В справочник «Графики работы» добавьте график работы, например 5/2.
Далее в форме списка нажмите кнопку "Заполнить графики работы". Откроется форма обработки заполнения графика работы.
В поле выбор периода выберите период (год) для заполнения графика. В поле «Выходные дни» через запятую перечислите порядковый номер выходного дня недели. Например: 6,7 (как суббота и воскресенье).
В поле «график работы» выберите ранее созданный элемент справочника «Графики работы». (если поле пустое)
Нажмите кнопку «Заполнить». 
В регистре сведений «График работы» проверьте корректность заполнения графика.

### 5
При необходимости в справочнике пользователи создайте нового пользователя. Или используйте уже имеющихся. 
На вкладке права доступа назначьте пользователю соответствующий должности профиль группы доступа.
Наименования профиля группы доступа  созданы для вашего удобства интуитивно понятными. Пример: для бухгалтеров профиль группы доступа - «БухгалтерИТФирмы», для кадровика – «Кадровик расчетчик».

### 6
Выдайте новые, поставляемые группы доступа пользователям, основываясь на занимаемыми ими должностями.

### 7 
1.Проинструктируйте бухгалтера о возможности автоматического формирования документов по реализации за месяц с помощью обработки «ВКМ массовое создание актов»;

2.Проинструктируйте менеджера о возможности создания договоров вида Абоненское обслуживание. Для них реализовано заполнение дополнительных полей: «дата начала» и «окончания договора», «стоимость часа работы», «Абоненская плата». Эти поля должны быть заполнены для корректного формирования документов реализации;
